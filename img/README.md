# Imágenes para Polaroids

Coloca aquí tus imágenes con los siguientes nombres:
- `foto1.jpg`
- `foto2.jpg`
- `foto3.jpg`
- `foto4.jpg`
- `foto5.jpg`
- `foto6.jpg`
- `foto7.jpg`

Las imágenes se mostrarán automáticamente en los polaroids de la página 4. 

Se recomienda que las imágenes sean cuadradas (150x150px mínimo) para un mejor resultado.
